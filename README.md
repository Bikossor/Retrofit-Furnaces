# Retrofit-Furnaces

This mod adds two recipies so you can craft Steel furnaces using Stone furnaces or Electric furnaces using Steel furnaces.

## Recipies:

- Steel furnace:
    - 6 Iron plates
    - 8 Stone bricks
    - 1 Stone furnace
- Electric furnace:
    - 4 Iron plates
    - 8 Stone bricks
    - 5 Advanced circuits
    - 1 Steel furnace
